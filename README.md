# Projet_meteo_drones
C'est un projet qui analyse les données météos de Parakou et vois les temps favorables pour les drones.Ces données ont permis de réaliser des graphiques avec le logiciel SPSS et de tirer des conclusions.
## Données
Source : openMeteo.com — Parakou, Bénin (5 ans)

## Outils utilisés
- SPSS (analyse statistique et graphiques)

## Résultats principaux
-un décollage de drones est recommandé dans la période allant du mois de Mai au mois de Décembre (périodes à vents faibles)
-Décollage de drones risqué dans la période allant du mois de Mars au mois de Décembre (période marquée par une forte humidité dans l'atmosphère)
